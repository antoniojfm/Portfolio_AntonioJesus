Proyectos

Proyecto Cooperativa del Pilar

Este proyecto trata de una pagina web para la Cooperativa del Pilar de Villacarrillo desarrollada en bootstrap. Esta pagina nos muestra información, imágenes y productos de la cooperativa.

Proyecto API Football

Este proyecto en grupo trata de una aplicacion en Android desarrollada en Kotlin que muestra estadisticas de partidos de futbol como resultados, goles, maximos goleadores, tabla de clasificacion.

Proyecto Administracion y Gestion de Usuarios

Este proyecto desarrollado en PHP, MySQL HTML y CSS y consiste en administrar usuarios desde una base de datos con creacion, listado, actualizacion y borrado.

Proyecto Web Ingeniería y Gestion

Este proyecto es una web desarrollada en HTML y CSS para informacion acerca de la empresa Ingeniería y Gestión de Villacarrillo - Jaén.

Proyecto App Ciudades

Este proyecto trata de una App hecha en Kotlin que nos permite añadir ciudades a un mapa y a la base de datos, tambien actualizar, borrar, listar y otras mas.

Proyecto API Rick-Morty

Este proyecto trata de una App hecha en Swift, que nos permite obtener datos de la API de Rick y Morty para obtener informacion de los personajes, episodios.
